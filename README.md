# Terminal Fruit Guesser 🍎

A classic command-line word guessing game built in Python.

## Description
This is a simple terminal game where the player tries to guess a hidden fruit name, one letter at a time. I built this project to solidify my understanding of core Python fundamentals before moving on to web development. 

**Concepts practiced:**
* `while` loops and game state management
* Using `set()` for efficient duplicate tracking
* Input validation and error handling
* Conditional logic

## How to Play
1. Make sure you have Python installed on your computer.
2. Download or clone this repository.
